You can browse the git repo in your browser here. To clone the source code, run the following commands: 

  git clone http://ongardie.net/www.git/
  git submodule update --init

Vim Configuration

I've posted the configuration files I use for vim/gvim in a public git repo.

You can browse the git repo in your browser here. To download and use my vim configuration, run the following commands: 

  cd
  git clone http://ongardie.net/vim.git/ .vim
  ln -s .vim/vimrc .vimrc
